# docs

This directory holds the human-readable source of truth for NOC.

## Subdomains
- `master-index/` top-level navigation and authority map
- `foundation/` workstation foundation, baseline, runbook, changelog
- `models/` model registry and related model governance docs
- `governance/` ontology, contracts, evals, failure modes, memory policy
- `rollout/` deployment sequencing and pilot rollout rules
- `runbooks/` operator procedures and incident handling
- `decisions/` architecture decisions and decision logs

## Rule
If a behavior is important enough to normalize, it should be documented here first or updated here the same day.
