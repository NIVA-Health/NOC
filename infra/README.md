# infra

This directory holds workstation and operational implementation assets.

## Intended contents
- `workstation/` baseline and machine-specific notes
- `scripts/` startup, shutdown, health-check, backup, and cleanup scripts
- `config-templates/` redacted templates for runtime and model config
- `backup/` backup instructions and manifests
- `diagnostics/` repeatable validation and troubleshooting helpers

## Rule
Infrastructure should be boring, repeatable, and recoverable.
